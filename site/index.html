H.323 for Ethereal

Preface

This project was done by me from February to June 2000 at Philips Business Communications in Hilversum. They asked me to create a network sniffer which could decode H.323 traffic. I changed the Q.931 dissector which was already used in Ethereal, I also changed the existing RTP and RTCP (RFC 1889) dissectors to suit my needs. I changed an existing ASN.1 compiler from the OpenH323 project to generate source to be used in Ethereal. Since March 2002 the H.323 is supplied as a plugin for Ethereal in stead of a binary.

If you want more information, checkout the Sourceforge site.
