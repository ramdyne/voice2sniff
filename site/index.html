Ethereal binary with H.323 support

Preface

This project was done by me from February to June 2000 at Philips Business Communications in Hilversum. They asked me to create a network sniffer which could decode H.323 traffic. I changed the Q.931 dissector which was already used in Ethereal, I also changed the existing RTP and RTCP (RFC 1889) dissectors to suit my needs. I changed an existing ASN.1 compiler from the OpenH323 project to generate source to be used in Ethereal.

At this site you can download Ethereal in two varieties. The source is available for those that run Linux or for those running Windows who want to know how it all works. For those who want to download a binary, I have only got a Windows version available.
